# KSP-Props
Arduino-controlled physical props for Kerbal Space Program Vessels

## Dependencies
* kRPC (KSP mod, installed thru CKAN)
* kRPC Python Module
* PyMata (Firmata protocol implementation in Python)
* Arduino board (Preferably UNO)
